## What to build

Detail the Selection Action Bar UX in the project's `.pen` design file. This slice is **HITL** — run the `/ui-design` flow, treat as a sibling to PRD-106.

Two states in scope:

1. **Idle (Marquee active, no Floating Selection)** — buttons: `Copy`, `Cut`, `Paste` (disabled variant when clipboard empty), `Delete`, `Deselect`.
2. **Floating Selection active** — buttons: `Done`, `Cancel`, `Copy`.

Scope:

- Compact (<600px) and medium+ (≥600px) layouts. Compact = icon-only; medium+ = icon + label. Wide+ (≥1024px) = tooltip on hover.
- Light + Dark variants per state.
- Positioning examples: default 8 px above Marquee top edge, fallback below, sticky-to-viewport-edge when both overflow.
- Touch-target sizing: every button meets 44 × 44 pt floor (invisible padding when icon-only).
- Icon choices: confirm Lucide icons for Copy / Cut / Paste / Delete / Deselect / Done / Cancel during the flow. Confirm whether `Deselect` uses a distinct glyph from `Cancel` (semantically different actions).
- Disabled state styling for `Paste` when the Selection Clipboard is empty.
- Hidden-mid-drag fade-out and fade-back-in timing.
- Reduced-motion variant (no fade).

Reference existing patterns:

- `ReferenceLayerPlacementOverlay` (PRD-106) for overlay positioning and design tokens.
- LeftToolbar button sizing (36 × 36 px tool-btn) for the small-button visual baseline.

Constraints:

- All design tokens come from the existing token system (`docs/design-system.md`); new tokens require explicit agreement during the flow.
- No floating-panel pattern — this is a transient contextual UI, not a layout panel.

### Key Decisions

- Idle state uses `Copy`, `Cut`, `Paste`, `Delete`, and `Deselect`; Floating Selection state uses `Done`, `Cancel`, and `Copy`.
- Compact layout is icon-only with 44×44 hit targets; medium+ layout adds labels; wide+ behavior uses tooltip-on-hover.
- Lucide icons are locked: `copy`, `scissors`, `clipboard-paste`, `trash-2`, `square-dashed-mouse-pointer`, `check`, and `x`.
- `Deselect` uses `square-dashed-mouse-pointer`, not `x`, so it stays visually distinct from Floating Selection `Cancel`.
- `Done` is the only accent-filled action; `Delete` uses danger text/icon color; disabled `Paste` preserves footprint with reduced opacity.
- Positioning is above Marquee by default, below on top overflow, and sticky to the viewport edge when both vertical placements overflow.
- Mid-drag hide uses 100 ms fade-out and 120 ms fade-back-in; reduced motion uses immediate hide/show.
